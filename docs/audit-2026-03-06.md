## Axe 2 — Design system & tokens

### Ce qui est OK

- Tokens bien definis dans `src/index.css` @theme : bg, fg, accent, muted, surface, border, success, warning, danger, info
- Light mode override propre via `[data-theme='light']`
- Tokens CSS supplementaires dans `:root` (accent-rgb, transitions, z-index)
- `cn()` utilise partout ou il y a des classes conditionnelles
- Tokens semantiques utilises dans la majorite des composants (bg-bg, text-fg, text-accent, etc.)

### Ameliorations recommandees

- **Couleurs hardcodees `rgba(196, 64, 64, ...)`** dans 6 fichiers au lieu d'utiliser `var(--color-accent-rgb)` :
  - `src/pages/NotFound.tsx:14` — rgba(196,64,64,0.04)
  - `src/features/particles/DynamicParticles.tsx:159,178,194` — rgba(196,64,64,...) x3
  - `src/components/ui/Button.tsx:22` — rgba(196,64,64,0.4)
  - `src/components/ui/Card.tsx:35` — rgba(196,64,64,0.03)
  - `src/components/layout/Header.tsx:32` — rgba(196,64,64,0.4)
  - `src/pages/Playground.tsx:776` — rgba(196,64,64,0.05)

  Seul `CursorGlow.tsx` fait correctement : `rgba(var(--color-accent-rgb), 0.3)`. Les autres devraient suivre ce pattern.

- **Playground swatches incorrects** — `src/pages/Playground.tsx:240` affiche `dark="#FF6B6B"` pour accent mais la vraie valeur est `#c44040`
- **`src/components/ui/Switch.tsx:50`** — `bg-white` hardcode pour le thumb du switch. Ne s'adaptera pas au light mode (fond #b0b0a8). Devrait utiliser un token ou `bg-fg`
- **`src/pages/Playground.tsx:61-65`** — Le composant interne `Copyable` utilise un template literal avec ternaire au lieu de `cn()`. Le className passe en prop ne beneficie pas de `twMerge()`

### Problemes critiques

- **Desynchronisation accent color** : L'accent a ete change de `#FF6B6B` a `#c44040` dans `index.css` mais les fichiers suivants referent encore l'ancienne valeur :
  - `.claude/decisions.md:11,20`
  - `.claude/agents/designer.md:20`
  - `.claude/rules/styling.md:38`
  - `.claude/commands/migrate.md:63`
  - `DESIGN_SYSTEM.md:7,59`
  - `docs/DEPENDENCIES.md:25`
  - `public/favicon.svg:2` (fill="#FF6B6B")
  - `src/pages/Playground.tsx:240` (swatch display values)

**Score: 6/10** (la desynchronisation de l'accent est un probleme systeme)
